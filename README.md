# 数据集说明 #
 1. 训练数据标签中的像素：
 	+ 1代表NCR&NET区域。
	+ 2代表ED区域。
	+ 4代表ET区域。
	+ 0代表其他区域。
 2. 解释：
	+ NCR: the necrotic（坏死）
	+ NET: the non-enhancing（非强调区域）
	+ ED: the peritumoral edema（瘤周水肿）
	+ ET: enhancing tumor
